## Installation

```bash

```

Create a `.env.local` file with a valid `REACT_APP_NETWORK_URL` (e.g. local BSC node or Infura alternative):

```bash
REACT_APP_NETWORK_URL="http://localhost:8545"
```

## Usage

```bash
$ npm run start
```
